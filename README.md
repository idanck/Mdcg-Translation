# תרגום מסמכי MDCG לעברית

תרגום נאמן למקור לעברית של מסמכי ה-EU MDCG — קבוצת תיאום מכשירים רפואיים של האיחוד האירופי (Medical Device Coordination Group).

## מטרת הפרויקט

מסמכי ה-MDCG הם הנחיות מחייבות/ממליצות שפרסמה הנציבות האירופית לצורך יישום תקנות ה-EU MDR (Regulation 2017/745) ו-EU IVDR (Regulation 2017/746). פרויקט זה מבקש להנגיש את המסמכים לדוברי עברית, בעיקר לאנשי רגולציה, יצרנים וגופי הערכה.

## עקרונות התרגום

- **נאמנות למקור** — הכוונה היא לתרגום קרוב ככל האפשר לטקסט האנגלי המקורי
- **עקביות מינוחית** — כל מונח מתורגם באופן אחיד לאורך כלל המסמכים
- **שמירה על מספור** — מספרי סעיפים, נספחים ופסקאות נשמרים כפי שהם
- **מונחי מקצוע** — מונחים שאין להם תרגום מקובל יופיעו באנגלית במקביל
