# תרגום מסמכי MDCG לעברית

תרגום נאמן למקור לעברית של מסמכי ה-EU MDCG — קבוצת תיאום מכשירים רפואיים של האיחוד האירופי (Medical Device Coordination Group).

מקור: [European Commission – MDCG Endorsed Guidance](https://health.ec.europa.eu/medical-devices-sector/new-regulations/guidance-mdcg-endorsed-documents-and-other-guidance_en)

---

## מטרת הפרויקט

מסמכי ה-MDCG הם הנחיות שפרסמה הנציבות האירופית לצורך יישום תקנות ה-EU MDR (Regulation 2017/745) ו-EU IVDR (Regulation 2017/746). פרויקט זה מבקש להנגיש אותם לדוברי עברית — אנשי רגולציה, יצרנים וגופי הערכה.

## עקרונות התרגום

- **נאמנות למקור** — קרוב ככל האפשר לטקסט האנגלי
- **עקביות מינוחית** — מונח מתורגם אחיד לאורך כלל המסמכים (ראה `glossary/glossary.md`)
- **שמירה על מספור** — מספרי סעיפים, נספחים ופסקאות נשמרים כפי שהם
- **מונחי מקצוע** — מונחים ללא תרגום מקובל מופיעים באנגלית במקביל

---

## מבנה הריפו

```
documents/
└── MDCG-YYYY-N/
    ├── README.md          ← שם, קישור מקור, סטטוס תרגום
    ├── original/          ← קובץ המקור האנגלי (PDF)
    └── translation/       ← התרגום לעברית
glossary/
└── glossary.md            ← מילון מונחים עקבי
```
